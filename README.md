## Getting Started

### Prerequisites

- Node.js 16+
- npm/yarn

### Installation

1. Clone the repository

```bash
git clone git@github.com:Zeyu-Chen/Webtube.git
cd Webtube
```

2. Install dependencies

```bash
npm install
```

3. Start the development server

```bash
npm run dev
```
